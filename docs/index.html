# AmbiLearn *L'éclairage d'ambiance qui apprend à s'adapter !* - Thomas **B**OURG - Julian **B**RUYAT - Théo **B**UTTEZ
## Concept Un éclairage qui s'adapte à la lumière ambiante
### La carte Arduino La carte Arduino possède : - Trois détecteurs de luminosié (rouge, vert et bleu) - Une led multicolore La carte détecte la luminosité de l'environnement et produit ou non un éclairage en fonction de l'ambiance demandée.
### Le site web - Choisir une ambiance - Déclencher l'allumage de la led - Noter l'interaction (valeurs de luminosité - éclairage) afin de s'adapter - Consulter la liste des interactions apprises
### Inventaire matériel - Carte Arduino UNO - Une led multicolore - Trois photorésistances - Trois gélatines (rouge, vert et bleu)
### Technologies #### Serveur NodeJS Serveur web avec *Express* Communication série avec *Johnny-five* #### Client web CSS avec Bulma
